### RUN

Software:
- Java: openjdk version "21.0.1" 2023-10-17 LTS
- Gradle: Gradle 8.7

If you use Gradle, you can run the application by
using `./gradlew bootRun`

Alternatively, you can build the JAR file
by using `./gradlew build` and then run the JAR file, as follows:

`java -jar build/libs/gs-rest-service-0.1.0.jar`
